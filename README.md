# gas-emissions-check
GasEmissionsCheck is a POC for a product that automatically measures gas analyzer data of a vehicle with engine and stores in database. 

The whole physical ecosystem includes:
- **Raspberry PI Zero 2 W**
- **Raspberry PI 5**
- Gas Analyzer **DEK DG-04 IR**
- External Smart Wi-Fi Camera **TP-Link Tapo C320WS**
